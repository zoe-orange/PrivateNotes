    4. Linux网络环境配置

    4.1 自动抓取

    缺点： 每次自动获取的ip地址可能不一样，不适用于做服务器

    4.2 指定ip地址

    1. 直接修改配置文件来指定IP，并可以连接到外网，编辑：vim /etc/sysconfig/network-scripts/ifcfg-eth0
    2. 重启网络服务：service network restart
    3. 重启系统：reboot
